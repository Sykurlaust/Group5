# GC Rentals Brand Guide

## Brand Colors
- Primary Blue: `#2A8DEB`
- Deep Blue: `#1C4ED8`
- Light Blue: `#BDEBFF`
- Accent Light: `#7EE7FF`
- Dark Text: `#111111`
- White: `#FFFFFF`

## Usage Rules
- Use the SVG files as the default on web wherever possible.
- Keep clear space around the logo equal to at least the width of the icon door.
- Do not stretch, skew, rotate, or recolor the logo outside approved variants.
- Use `gc-logo-navbar.svg` in site headers for best readability and sizing.
- Use `gc-logo-icon.svg` where text is too small or unnecessary.
- Use white and black variants only on high-contrast backgrounds.

## Web Implementation Note
SVG is the main format for web UI because it stays sharp on all screens and scales without quality loss.
